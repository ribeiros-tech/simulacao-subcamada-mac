# Explicação Teórica – Subcamada MAC

Este projeto demonstra o funcionamento dos principais protocolos de controle de acesso ao meio utilizados na subcamada MAC da camada de enlace.

## Protocolos estudados
- ALOHA: acesso aleatório, alto número de colisões
- CSMA/CD: escuta do canal e detecção antecipada de colisões
- Token Ring: acesso ordenado por passagem de permissão

## Conclusão
Protocolos com acesso ordenado apresentam melhor eficiência, porém maior complexidade.
